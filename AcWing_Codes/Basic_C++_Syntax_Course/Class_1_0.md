## 一、AcWing课程概况和层次结构

整个课程有`4`个level.

* level`1`：语法基础课
   介绍C++在算法竞赛或者考试中的语法；
* level`2`：算法基础课
   介绍常用的算法和算法原理(包括证明)，以及算法模板；
* level`3`：算法提高课
   各种常用算法的应用技巧；
* level`4`：算法进阶课
   讲解难题和复杂的题.
